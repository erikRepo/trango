# trango developer documentation

This is the developer-facing documentation for trango (product name
**TrangoPlayer**), a Rust + Slint + libmpv language-learning video player.

For the product handoff spec (views, states, design tokens) see the
repository root `README.md`. For the development workflow (TDD, scripts,
git workflow, Rust conventions) see the repository root `CLAUDE.md`. For the
step-by-step development roadmap see the repository root `TODO.md`.

This book covers:

- **Usage** — installing and running trango
- **Architecture** — crate structure, state model, subtitle pipeline
- **Specs** — functional specifications for app behavior
- **Technology** — one page per notable dependency, why it was chosen and
  how it's used in this project
